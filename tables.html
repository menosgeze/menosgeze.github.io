My research was to compute the Tate Shafarevich group order and type of as many rational elliptic curves as possible assuming standard conjectures. Here, I present you the results of my computations. I also present some results on the p-adic height of rank 1 rational elliptic curves.

Table 1: Rank 0 rational elliptic curves

I consider all rank 0 elliptic curves in Cremona's database and these up to conductor one million in Stein Watkins' database. These databases already contain the Tate Shafarevich group order. Using descent methods in Magma, computational algebra system, and Tom Fisher's results from 5 descent, I found the Tate-Shafarevich group invariants listed here, under the assumption of the Birch and Swinnerton Dyer conjecture and the Generalized Riemann Hypothesis (for computing ideal class groups.)

  • The column headings are: conductor, a-invarinats, Ш analytic order, Ш group invariants.

Table 3: Rational elliptic curves of rank larger than 1

I considered all elliptic curves in Cremona's database and these up to conductor one million in Stein Watkins' database of conductors 2, 3 and 4. These databases do not contain the Tate Shafarevich group order of these curves. Using point search methods in SageMath (which uses Michael Stoll's ratpoints module in PARI/GP) and point search methods in curves obtained by Magma's descent methods, I found the generators of most curves. Using a Diophantine inequality, I was able to determine the Ш group order of several rank 2 elliptic curves where I did not find the a second generator of the Mordell Weil group.

  • The column headings for the file of rank 2 elliptic curves are: conductor, a-invarinats, Ш analytic order, Ш group invariants, 1st generator, 2nd generator. For the rank 3 and rank 4 elliptic curves, there are two more entries that contain the rest of the generators. The column headings for the file of rank 2 elliptic curves where the second generator is not known are: conductor, a-invariants, Ш analytic order, Ш group invariants, a non-torsion generator, the group ranks of the 2-torsion and 3-torsion parts of ⠘, and the naive logarithmic height bound for the point search performed in SageMath.
